# 本多忠勝

**角色定位**:鬥士  
**適合走線**:A、C  
**角色強勢期**:中後、後期  

## 角色簡介
實質上為此遊戲後期可以carry的角色其一，  
T+D的攻速，  
R的擴散傷害，  
是可以作為老大的重點。  
W的無傷則是大幅增加自己的生存率及延續輸出的手段。  
只需要注意是以拆塔還是殺人為重即可。  

總結來說，  
就是一個**前期鬼農，中後期當偷拆狗或是核心輸出的角色**，  
非常不推薦新手上手，  
對於雙方角色差異、裝備成型、傷害計算、防御裝備理解不深的時候，  
他前期的機體足以讓你痛苦到遊戲結束。  

---

## 💎 出裝（走C） 

**出門：** 鱗片甲 + 褲子 + 短甲 + 軍糧丸  

鱗片甲跟褲子無須解釋，  
短甲是我研究在1等的時候可以最高CP值減免物理傷害的裝備，  
400元可以讓你50%機率-50傷害，  
以強盜前期的數量及造成的傷害可以省下好幾百的血量。  
並且軍糧丸可以進一步在利用R快速刷野的時候提升生存率，  
因為本多前期真的沒什麼反抗的本錢。  

---

## ⚔️ 後補裝備  

**暗部 + 武尊環 + 真知短甲(走C)**  

為了彌補前期的弱勢程度以及刷野速度，  
**暗部提升了跑速+魔抗+攻擊力，**  
要強調的是，  
因為R的關係，  
基本上**攻擊力越高=刷野越快=風險越低**，  
武尊環就是為了進一步提升刷野速度的裝備，  
而且如果走上C，  
**甚至可以不用先出暗部肉身去買，**  
就不用花買馬的錢，  
後面可以合紅雪。  

真知清忍其實就是為了保住自己的發育範圍。

## ⚔️ 中期裝備  

1.**野太刀 + 神直**  
2.**野太刀 + 鬼切 + 逆刃**  
3.**野太刀 + 斷簡** 


這個時間點其實有非常多的選項。  
先講大方向:  
由於可以上合紅雪增加拆塔速度以及傷害，  
野太刀基本上是必出。  
因為在我的計算下，  
家裡的高階武器之中，  
**紅雪的輸出(對塔or人)期望值是最高的。**  

其次，  
光是有野太刀提供的攻速以及爆擊，  
有大招可以配合隊友做一些簡單的擊殺，  
也就是說**光是一把野太刀，近戰傷害就已經很高了**  
只缺你的兄弟幫你控場。  

另一方面，  
可能會有人對於攻速有武士服的需求，  
但是實際上暗部+野太刀/鬼切其中一把+逆刃就已經接近了逆刃連暈的效果，  
暗部+野太刀已經具備了單殺的能力，  
本多的等級在足夠的情形下，  
光靠TD就攻速就已經足夠，  
**武士服只是為了增加回血和魔量的拖裝選擇。**  

接下說明三種路線:  
1.**野太刀 + 神直**  
這個是兼具體質(會戰/拆塔)以及輸出的通用套路。  
神直提供的50魔力追擊+體質+些微攻擊力，  
其實就已經可以在會戰做出很不錯的傷害。  
你可能會問:  
阿不是要逆刃?  
逆刃是**在單殺情形、或是隊友睡著了**時候的選擇，  
當你實際下場打，  
你會發現**會戰本多最缺的只有追擊能力。**  
而且逆刃**並沒有提供任何護甲或是血量**，  
花了3300輸出可能只比野太刀多一點。  

2.**野太刀 + 鬼切 + 逆刃**  
這個就是你們想看到的輸出連暈套，  
就是"犧牲進場/追擊能力，提升單殺能力"的本多。  
如果你不信任隊友，  
對面的角色也不會瘋狂風箏，  
那是可以走這套，  
但是靈活性我認為不如1.。  
至於法球，  
你要出凍雲和雷切可以，  
無非就是貴1500左右，  
對於本多吃緊的經濟以及後續可以上合的夜哭要斟酌。  

3.**野太刀 + 斷簡**  
光看裝備就不難理解，  
這套並不是拿來打團的(當然如果對面有高物攻體系，要稍微參團可以)。  
基本上就是偷拆流，  
增加防禦以及跑速拚偷拆，  
但是**這個遊戲要玩偷拆必須要有非常良好的地圖觀、裝備理解、獨立思考能力**，  
新手建議就是走1.2.打團榮錯率比較高。  

因為你們不會知道護甲要多少可以扛堅守拆塔、人家什麼時候會來抓你拆塔、不能被隊友的言語影響拆塔意識。

## 🛡️ 中後期成形

1.**紅雪 + 神直 + 鬼切/逆刃**  
2.**紅雪 + 鬼切 + 逆刃 + 神直**  
3.**紅雪 + 斷簡 + 嗜血妖刀**  

上合紅雪我懶得多提，  
反正就是家裡裝備中輸出期望值最高的刀。  
1.**紅雪 + 神直 + 鬼切/逆刃**  
你可能會問，  
為啥不用鬼切和逆刃都出?  
當然是可以，  
但其實光是有紅雪和神直，  
**已經可以打出很可觀的爆發傷害，**  
剩下就是鬼切補傷害或是逆刃補單殺控場，  
就看對面的出裝及角色對應。  
這個時間點其實如果優勢，  
就已經可以衝名刀，  
如果劣勢就還是出完整套吧。  

2.**紅雪 + 鬼切 + 逆刃 + 神直**  
這個就是1.說的完整套，  
把單控+追擊+傷害補齊，  
也是路人最不會挑剔的一套，  
**不想被嘴出裝就這樣出完。**  

3.**紅雪 + 斷簡 + 嗜血妖刀**  
**這套其實就是衝800和1300的裝備。**  
1300光這樣還不夠，  
需要補硬度或是瞬戒防止小兵卡你，  
吃掉你W無敵時間和斷簡的減傷時間。  
但是800是絕對穩妥而且不殘血，  
敵人顧不到你的時候也可以一路拆到內塔。  
需要注意的是，  
**800和1300不提供嗜血妖刀的吸血**，  
你只能敲小兵吸回來。

## 🔥 後期裝備  

**紅雪/長傳 + 蜻切 + 虎撤 + 神直+陸奧 + 夜哭/村正 + 御神(自己挑五把)**  

名刀選擇其實彈性十足，  
**但是第一把是讓你有重大提升的關鍵。**  
補陸奧就是強化偷拆靈活性，  
補御神就是提升會戰生存率及輸出時間，  
補虎撤就是提升連暈能力和傷害，  
補蜻切就是同時強化拆塔能力和會戰跑速，  
換村正就是強化拆塔能力和會戰生存率。  
一切看你想要的是什麼、面對什麼情形。  

以成型來說，  
要走爆發輸出，  
紅雪+神直+御神+法球+虎撤足矣。  
要走拆塔，  
紅雪+村正+斷簡+御神+蜻切可以隨便拆。  
阿要綜合當貪狗，  
陸奧+紅雪+御神+法球+虎撤也差不多。

## 🧠 配點順序  

**REW=>WW(BB)T=>W(B)**   
- E最後滿，因為R有提升D的效果   
- 怕法傷先滿W，對面沒法傷先滿B

## 🪄 技能使用與實戰細節  

殺人要件:
E中+野太刀

接技順序:  

近身(走路/神直)+E+抗+T+D  
有緩到就可以全開，
人家跑了就神直。  

如果是走上C，  
**出門可以開神速戰法搶軍師，**(有人嘴你不用理他，你是老大)  
擴散傷害可以比其他人速洗，  
但如果對面有忍角，  
你開神速跟他隱形跑一樣快就不用開，也別搶軍師。  
搶軍師有個細節，  
如果是聯合軍，  
**上面的紅色強盜閃避機率高，**  
**會大幅降低清野速度，**  
拉出來敲軍師。  
織田軍的話直接敲軍師下面的強盜即可。  

走下C就不用開任何戰法，  
全部刷一刷即可。  
擺忍的話擺野怪旁邊保發育即可。  

要走A也行，  
可以買回魔鋼和再生鋼保持續戰力，  
**遇到平手或是淺井高法傷的可以出制魔腹捲/護符，**  
他們會對殺你失去興趣。  
因為走A你注定劣勢被推線或控線，  
要非常積極給隊友情報和支援隊友，  
否則壓力會跑到C上。
